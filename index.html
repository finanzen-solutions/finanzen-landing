Preguntas frecuentes

¿Es Finanzen gratis?

Sí, Finanzen ofrece un plan gratuito con todas las funciones esenciales para gestionar tus finanzas personales. También contamos con planes premium con características avanzadas.

¿Puedo gestionar mis gastos en diferentes monedas?

Sí, Finanzen soporta múltiples monedas incluyendo dólares y bolívares. Puedes configurar tu moneda principal y agregar gastos en cualquier moneda.

¿Cómo se actualiza la tasa de cambio entre monedas?

Las tasas de cambio se actualizan automáticamente cada día usando fuentes confiables. También puedes configurar una tasa personalizada si lo prefieres.

¿Mis datos financieros están seguros?

Absolutamente. Utilizamos encriptación de nivel bancario (AES-256) para proteger toda tu información. Tus datos nunca se comparten con terceros.

¿La aplicación funciona sin conexión a internet?

Sí, puedes registrar gastos sin conexión. Los datos se sincronizarán automáticamente cuando vuelvas a tener internet.
